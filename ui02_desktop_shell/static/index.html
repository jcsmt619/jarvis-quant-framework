    Architecture
    Python standard-library launcher, local UI service, read-only same-origin gateway, and installed browser app window.
    Local only
    Services bind to 127.0.0.1. The CLI, research engines, tests, automation, Cline, and Codex remain independent.
    Sources
    Fixture mode and sanitized recorded-response mode are supported. Provider validation remains pending.
    Shutdown
    Closing the desktop app stops only UI-05-owned UI processes and removes the temporary browser profile.
    Logs
    Bounded sanitized operational logs are stored under local application data outside Git.
    Shortcuts
    Shortcuts are created only by explicit operator installer invocation.
    Stale lock
    Run the launcher inspect-lock or clean-shutdown command to review or remove stale UI-05 locks.
